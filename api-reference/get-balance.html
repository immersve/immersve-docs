Get balance​

This endpoint returns the balance and available balance for a given asset.

Responses
  • 200
  • application/json
  • Schema
  • Example (from schema)
Schema
  • walletAddress string

    The blockchain address of the asset holder

  • balance string

    The amount held in the Immersve smart contract on behalf of the wallet address

  • availableBalance string

    The sum amount of funds thats are subject to a fund lock

  • chainId string

    Chain ID (see https://chainlist.org/)

  • currencySymbol string

    The symbol of the respective coin being used (currently only USDC)
