KYC

📄️ Submit KYC Statement

Submit a KYC statement about the cardholder.

📄️ Get KYC Statement

Retrieve the KYC statement for the cardholder if it exists.

📄️ Get KYC Profile

Get a KYC profile by account id if it exists.

📄️ Get Expected Spend Amount

Get the expected spend amount for the given cardholder.

📄️ Update Expected Spend Amount

Update the expected spend amount for the given cardholder.

📄️ Get Residential Addresses

Retrieve previously submitted residential addresses of the cardholder.

📄️ Submit Residential Addresses

Submit the current residential address of the cardholder in either structured or unstructured format.
